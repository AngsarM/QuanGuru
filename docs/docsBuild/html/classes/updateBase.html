updateBase module¶

This module contains the updateBase class, which is inherited by _sweep and Update classes.

class updateBase(**kwargs)[source]¶

Bases: qTools.classes.QUni.qUniversal

This class has only two attributes, which are private attributes intended to be used with different properties in the _sweep and Update classes to enable different terminologies for the same functionality.

key or _updateBase__key or __key

This string value is the name of an attribute for the objects in subSys dictionary of an instance of updateBase, and the value of that attribute is updated by an update function.

Type

str

_updateBase__function or __function

This is the aforementioned update function. This is not completely implemented in Update class, but the _sweep class, which will also be updated, has its basic use, which is pointing it to a default method to be used in updating the relevant attributes. The goal of making this an attribute is to enable user defined functions to be used, if some form of customisation is needed. save()[source]¶

This method extends the save of qUniversal by using the extended toBeSaved list and also collecting some of the information under a single key.

property key¶

The key property:

  • getter : returns _updateBase__key

  • setter : sets _updateBase__key string

  • type : str

property system¶

The system property wraps subSys dictionary to create new terminology, it basically:

  • getter : returns subSys[0] if there is only one item in it, else returns list(subSys.values()).

  • setter : works exactly as subSys setter.

_runUpdate(val)[source]¶

This simple method sets the attribute (for the given key) of every subSys to a given value (val). It returns None.
