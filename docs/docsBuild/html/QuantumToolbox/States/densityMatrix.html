densityMatrix Function¶

densityMatrix(ket: Union[scipy.sparse.base.spmatrix, numpy.ndarray]) → Union[scipy.sparse.base.spmatrix, numpy.ndarray][source]¶

Converts a ket state into a density matrix.

Keeps the sparse/array as sparse/array.

Parameters

ket (Matrix) – ket state

Returns

density Matrix

Return type

return: Matrix

Examples

>>> ket = basis(2, 0)
>>> mat = densityMatrix(ket)
(0, 0)      1

>>> ket = basis(2, 0, False)
>>> mat = densityMatrix(ket)
[[1 0]
[0 0]]

>>> ket = superPos(2, [0,1], sparse=False)
>>> mat = densityMatrix(ket)
[[0.5 0.5]
[0.5 0.5]]

>>> ket = superPos(2, {0:0.2, 1:0.8}, sparse=False)
>>> mat = densityMatrix(ket)
[[0.2 0.4]
[0.4 0.8]]
