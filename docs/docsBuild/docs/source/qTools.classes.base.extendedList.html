qTools.classes.base.extendedList¶

class extendedList(iterable=(), /)[source]¶

This class extends the built-in list class. It is introduced to be used with toBeSaved lists of every class that inherits from qUniversal. These lists contain the keys for the attributes to be saved (currently into a txt and hdf5 attributes). toBeSaved is a class attribute and extending it through the inheritance tree is achieved by extendedCopy method of this class.

Current saving methods are going to be improved, so, in future, this class might not be needed.
